# README

* Ruby version

* System dependencies
  - gem "rspec-rails"
  - gem "simplecov"
  - gem "capybara"
  - gem "launchy"
  - gem "shoulda-matchers"

* Configuration

* Database creation
  - `rails db:migrate`

* Database initialization

* How to run the test suite
  - `bundle exec rspec`

* Deployment instructions
  1. Clone down repo with `git clone git@github.com:zach-bergman/relational_rails.git`
  2. Navigate into project with `cd relational_rails`
  3. `rails s` to start local server
  4. `http://localhost:3000/` in browser
